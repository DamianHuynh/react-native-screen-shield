# react-native-screen-shield

Prevent Screenshot Image in your iOS/Android app.

## Installation

```sh
npm install react-native-screen-shield
```

## Usage

```js
import { ScreenShieldView } from "react-native-screen-shield";

// ...

<ScreenShieldView color="tomato" />
```
